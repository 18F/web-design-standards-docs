Responsive variants

Add a responsive breakpoint prefix separated with a : to target a utility at a responsive breakpoint and higher, following a mobile-first methodology.

Example

.tablet:padding-y-2

Output

@media screen and (min-width: 640px) {
  .tablet\:padding-y-2 {
    padding-left: 1rem;
    padding-right: 1rem;
  }
}

Set which breakpoints are available to USWDS utilities with the $theme_output_breakpoints variable found in _project-utilities-settings.scss.
